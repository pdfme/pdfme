FAQ

  • Can I use this for free?

    Yes. Free and unlimited.

  • The text on the pdf is garbled...

    You can only use the alphabet in this app. If you want to use other characters, you have to load other font.

  • Can I access the source code?

    Source code is available at https://github.com/pdfme/pdfme/tree/main/website/src/pages/demo.
